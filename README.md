# 오늘부터진짜

## 개요
- 카카오 푸시 알림을 이용해 오늘의 할 일을 잊지않고 끝낼 수 있게 도와주는  투두리스트.

## 주요기능
- 오늘의 할 일 추가/변경/삭제
- 중요한 일 설정
- 완료한 일 설정
- 오늘해야할 일과 오늘까지 남은 시간 카카오 푸시알림
e.g. 종료까지 n시간 남았습니다 / 할 일 1, 할 일2, 완료한 일1

## 기술
- Client: React
- Server: MongoDB
- open API: kakao push notification
